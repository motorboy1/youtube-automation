## 2. 무료 웹 호스팅 및 레이드라이브 설정 (코딩-배포 동시 트릭) 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/UGZTzPS2VZU/86.jpg)
1. **무료 홈페이지 호스팅 신청 및 정보 확보** 
  1. 다(Da)에서 무료 홈페이지 호스팅을 신청하며, 6개월 사용 후 연장이 가능하여 무기한 무료 사용이 가능하다. 
  2. 신청 완료 후 다음 정보들을 반드시 메모해야 한다. 
  1. **FTP 정보**: FTP 아이디, 비밀번호
  2. **DB 정보**: DB명, DB ID, DB 비밀번호
  3. **환경 정보**: PHP 버전
  4. **접속 정보**: 홈페이지 주소, IP 정보
  3. 이 정보들은 나중에 클로드에게 알려주어 DB 작업 및 코딩에 활용하게 된다. 

2. **레이드라이브****(Raidrive) 설치 및 연결** 
  1. 레이드라이브는 FTP 서버를 내 컴퓨터의 폴더처럼 쉽게 사용할 수 있게 해주는 **무료 프로그램**이다. 
  2. 레이드라이브를 검색하여 다운로드 및 설치를 완료하고, 구글 계정으로 로그인하면 편리하다. 
  3. 레이드라이브에서 '추가'를 클릭하고, 나스 선택 후 FTP를 눌러 홈페이지 계정을 연결한다. 
  4. 메모해 둔 홈페이지 IP 주소와 FTP 아이디, 비밀번호를 입력하고 연결 버튼을 누르면 설정이 완료된다. 
  5. **코딩-배포 동시 트릭**: 연결이 완료되면 내 컴퓨터에 Z드라이브가 생성되며, 이 드라이브 안의 HTML 폴더가 홈페이지 파일이 있는 곳이다. 
  1. 클로드가 이곳에 파일을 만들면 **즉시 홈페이지에 반영**되어 코딩과 배포가 동시에 이루어진다. 


## 3. 클로드 데스크톱 MCP 기능 최적화 및 설정 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/UGZTzPS2VZU/170.jpg)
1. **클로드 데스크톱**** ****프로젝트 지침**** 및 MCP 활용** 
  1. 클로드 데스크톱의 프로젝트에 지침을 입력하여 작업을 시작하며, 이때 MCP(Multi-tool Control Panel) 도구를 사용한다. 
  2. 최근 업데이트 이후 MCP는 망치 아이콘이 아닌 **툴 아이콘**에서 찾을 수 있으며, 원하는 기능을 켜거나 끌 수 있다. 

2. **주요 MCP 기능 목록 및 역할** 
  1. **컨텍스트(Context)**: 코딩 오류를 방지해 주는 역할을 한다. 
  2. **텍스트 에디터(Text Editor)**: 파일 시스템 MCP의 고질적인 에러를 해결하고 **토큰을 절약**해 준다. 
  3. **구글 서치(Google Search)**: 구글에서 자료를 직접 검색한다. 
  4. **플레이라이트 스텔스(Playwright Stealth)**: 접근하기 어려운 웹페이지도 잘 보여준다. 
  5. **터미널(Terminal)**: 컴퓨터 명령어를 실행할 수 있다. 
  6. **이미지 생성(Image Generation)**: 최신 **GPT-4**를 사용하여 이미지를 자동으로 만들며 품질이 매우 뛰어나다. 

3. **텍스트 에디터 MCP****의 핵심 이점** 
  1. 텍스트 에디터는 Z드라이브에 바로 파일을 생성하며, 이 덕분에 파일이 만들어지는 즉시 다(Da) 호스팅 홈페이지에 반영된다. 
  2. 이 기능은 매우 편리하며 시간을 획기적으로 절약해 준다. 


## 4. 풀 에이전트의 웹사이트 기획 및 개발 과정 


### 4.1. 리서치, 이미지 생성 도구 제작 및 프로젝트 관리 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/UGZTzPS2VZU/237.jpg)
1. **리서치 및 이미지 생성 계획** 
  1. 클로드는 구글 검색과 플레이라이트 스텔스 MCP를 사용하여 관광 사이트를 리서치하고, 그 결과를 문서로 저장한다. 
  2. 최신 GPT-4 이미지 모델 MCP를 사용하여 멋진 이미지를 직접 생성한다. 
  3. 웹 이미지 크기는 정해져 있지 않고 매번 다를 수 있으므로, 클로드가 **이미지를 원하는 사이즈로 잘라주는 프로그램**을 직접 만들게 한다. 
  1. 이 도구는 필요할 때 클로드가 스스로 호출하여 사용하게 된다. 

2. **프로젝트 계획 문서 생성 및 활용** 
  1. 프로젝트 계획 문서를 만들어 전체 과정을 쉽게 관리하며, 작업 진행 시마다 내용을 업데이트하여 진행 상황을 정리한다.
  2. 이 파일이 있으면 중간에 채팅창 제한이나 에러가 발생해도 새 창에서 작업을 이어서 할 수 있어 편리하다.

3. **호스팅 서버 직접 접근 및 배포 자동화** 
  1. 클로드는 Z드라이브(웹 호스팅 연결 폴더)에 직접 접근하여 파일을 저장한다. 
  2. 이는 터미널과 텍스트 에디터 MCP를 통해 가능하며, 호스팅 서버에 파일을 생성, 수정, 삭제까지 할 수 있다. 
  3. 클로드가 파일을 생성하자마자 바로 웹에 배포되는 시스템이 구축된다. 


### 4.2. 웹 분석, 코딩, 백엔드 작업 및 토큰 절약 전략 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/UGZTzPS2VZU/318.jpg)
1. **유명 관광 웹사이트 분석 및 ****디버깅** 
  1. 클로드는 화면 오른쪽 브라우저를 통해 유명 관광 웹사이트를 스스로 분석한다. 
  2. 플레이라이트 스텔스 MCP는 페이지 분석, 코드 체크, 스크린샷 촬영을 수행하며, 콘솔 로그로 오류 체크를 하여 **혼자 ****디버깅**까지 할 수 있다. 
  3. 웹 리서치가 끝나면 상세한 기획이 시작된다. 

2. **프런트 엔드 및 이미지 생성** 
  1. 클로드는 HTML, CSS, 자바스크립트를 이용해 메인 페이지를 직접 만든다. 
  2. 필요한 이미지를 스스로 생성하며, 이미지 생성 프롬프트도 혼자 만들고 이미지를 직접 만들어 적절한 위치에 저장까지 완벽하게 수행한다. 

3. **백엔드**** 및 DB 작업 자동화** 
  1. 클로드는 백엔드인 PHP 코드도 작성하고 데이터베이스 작업도 가능하다. 
  2. 미리 프로젝트 지침에 넣어 둔 DB명, ID, 패스워드, PHP 버전을 기반으로 클로드가 스스로 DB 작업과 PHP 코드를 생성한다. 
  3. 이 과정은 기획, 디자인, 프론트 엔드, 백엔드, 배포까지 아우르는 **풀 자동 에이전트**의 역할을 보여준다. 

4. **텍스트 에디터 MCP****를 통한 토큰 절약 전략** 
  1. 텍스트 에디터가 파일을 읽을 때 전체 파일을 읽지 않고 **20번째 줄만 골라 읽는** 등 꼭 필요한 부분만 읽어 토큰을 절약한다. 
  2. 이는 커서 AI의 시스템 방식이며, 토큰 절약은 더 많은 작업을 가능하게 한다. 
  3. 파일을 만들 때 한 번에 너무 길면 에러가 발생하므로, 처음에는 조금씩 만들고 나중에 추가하도록 지침에 넣어 에러 발생 시에도 작업을 이어 붙일 수 있게 한다. 


## 5. 실시간 디버깅, 기능 구현 및 최종 완성 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/UGZTzPS2VZU/425.jpg)
1. **웹 브라우저를 통한 실시간 결과 확인** 
  1. 웹 브라우저에서 도메인만 입력하면 홈페이지가 바로 나타나며, 세계 어디에서든 접속 가능하다. 
  2. 홈페이지의 이미지들은 클로드가 직접 만든 것이므로 저작권 걱정이 없다. 

2. **클로드의 디자인 문제점 자가 진단 및 수정** 
  1. 교통 정보 페이지 작업 중 디자인이 이상해지자, 사용자가 직접 설명하지 않고 클로드에게 화면을 캡처하게 했다. 
  2. 클로드는 스스로 스타일 충돌과 배치 이상을 문제점으로 찾아내고 결론을 내린다. 
  3. 이후 스스로 코드를 수정하기 시작하며, 디자인을 말로 설명할 필요 없이 스스로 확인하고 수정하여 버그를 해결한다. 
  4. 수정 과정을 거치며 디자인이 점점 깔끔해진다. 

3. **GPT-4**** 이미지 퀄리티 및 자동 저장 기능** 
  1. GPT-4의 이미지 퀄리티는 매우 뛰어나다. 
  2. 여행팁 메뉴에 이미지가 빠져 있을 때, 필요한 이미지를 만들도록 지시하면 이미지 여러 개를 한 번에 만들고 **적절한 위치에 자동 저장**까지 해준다. 
  3. 이를 통해 보기 좋은 여행팁 페이지가 완성된다. 

4. **DB 연동 및 검색/필터링 기능 구현** 
  1. 관광지 페이지 작업 시, 초기에는 HTML로만 되어 있던 것을 클로드가 데이터를 DB에 넣고 PHP로 연결한다. 
  2. 클로드는 파일 하나를 직접 만들어 필요한 내용을 DB에 넣는 작업을 혼자서 처리한다. 
  3. 완성된 페이지는 **검색 및 필터링 기능**까지 완벽하게 작동한다. 
  4. 클로드는 페이지마다 기능을 발전시키며 홈페이지를 완성해 나간다. 

5. **최종 결과 및 경제성 분석** 
  1. 완성된 홈페이지는 품질이 매우 뛰어나며, 수백만 원에 달하는 맞춤형 홈페이지 제작비를 클로드가 초스피드로 절약해 주었다. 
  2. 이 웹사이트는 단순 디자인뿐 아니라 **진짜 프로그램 기능**까지 작동하고 있다. 
  3. **데이터베이스**** 확인**: 다툼(Datom)을 통해 데이터베이스 조회 페이지로 들어가면 실제로 테이블과 데이터가 존재함을 확인할 수 있다. 
  1. 여기서 한 줄을 추가하면 홈페이지에 바로 나타난다. 
  2. 관리자 페이지도 쉽게 추가할 수 있다. 
  4. 이렇게 만들면 유지비가 거의 들지 않아 매우 경제적이다. 


## 6. 풀 에이전트 환경 구축을 위한 필수 설정 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/UGZTzPS2VZU/557.jpg)
1. **MCP 사용을 위한 필수 환경 설치** 
  1. MCP를 사용하려면 먼저 **Node.js**와 **Python**이 필요하며, 공식 홈페이지에서 다운받아 설치해야 한다. 
  2. MCP는 윈도우의 경우 명령 프롬프트에서 미리 설치해야 하며, 설치 명령어는 댓글로 제공된다. 

2. **클로드 데스크톱**** 설정 및 ****API 키**** 발급** 
  1. 설치 후 설정 내용을 클로드 데스크톱의 컨피그(Config) 파일에 붙여 넣어야 한다. 
  2. API 키를 넣는 곳에는 직접 발급받은 키를 넣어야 하며, 발급 방법은 댓글을 참고해야 한다. 

3. **프로젝트 지침****서의 중요성 및 내용** 
  1. 프로젝트 지침이 가장 중요하며, 특히 신상인 이미지 생성 MCP는 클로드가 헷갈릴 수 있으므로 지침서가 필수적이다. 
  2. 지침서에는 다음 정보들이 포함되어 있다. 
  1. 홈페이지 도메인 및 소스 폴더 위치
  2. DB 정보 및 PHP 사용 여부
  3. 텍스트 에디터 MCP 사용 시 자주 생기는 오류 방지법
  3. 이 지침은 직접 사용하며 최적화한 내용이므로 사용자의 시행착오를 크게 줄여준다.
