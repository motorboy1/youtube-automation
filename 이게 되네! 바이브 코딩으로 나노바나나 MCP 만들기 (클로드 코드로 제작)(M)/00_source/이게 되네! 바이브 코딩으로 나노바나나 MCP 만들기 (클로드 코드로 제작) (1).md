## 📌 바이브 코딩으로 나노바나나 MCP를 만드는 방법은?
 GPT-5 딥리 서치로 자료를 수집하고, 클로드 코드에 PDF를 던져 계획 및 구조를 짠 뒤, <mark>바이브 코딩으로 MCP를 개발하고 GPT-5의 분석 능력으로 에러를 해결</mark>하며 완성합니다. 


## 💡 나노바나나 MCP의 주요 기능은?

- 이미지 생성 및 편집
- 여러 이미지 합치기
- 서버 상태 확인 및 예상 비용 안내
- 스타일 프리셋 적용

이 콘텐츠는 **GPT-5와 클로드의 협업**을 통해 최신 이미지 생성 모델인 나노바나(Gemini 2.5 Flash Image API)를 활용하는 **완전 자동화 MCP(Master Control Program) 제작 과정**을 상세히 공개합니다. 며칠 밤을 새워야 할 코딩 작업을 AI에게 맡겨 단 몇 분 만에 이미지 생성, 편집, GIF 제작, 심지어 동화책 제작까지 가능한 **'바이브 코딩'의 실용적 방법론**을 배울 수 있습니다. AI를 단순한 도구가 아닌, **역할을 부여하여 성능을 극대화**하는 디버깅 꿀팁과 함께, 제작된 MCP를 구독자에게 배포하는 과정(보안 검토 및 설치 방법)까지 담고 있어 AI 자동화 시스템 구축에 관심 있는 분들에게 구체적이고 즉각적인 적용점을 제공합니다.
## 1. 나노바나 MCP 제작 배경 및 목표 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/zklpQ5NlSfk/0.jpg)
1. **나노바나(Gemini 2.5 Flash Image API)의 인기와 활용** 
  1. 최근 챗지피티 이미지에 이어 나노바나가 이미지 생성 분야에서 대세로 떠오르고 있다. 
  2. 나노바나는 일관성이 뛰어나 피규어 제작, 가상 옷 입히기 등 다양한 방식으로 활용되고 있다. 

2. **MCP(Master Control Program) 제작 아이디어 제시** 
  1. 나노바나를 직접 사용하는 대신, 클로드(Claude)에게 나노바나를 활용하는 MCP를 만들도록 시키는 아이디어를 구상했다. 
  2. MCP로 완전 자동화 시스템을 구축하면 며칠 밤을 새워야 하는 코딩 작업 없이도 즉시 기가 막힌 결과물을 얻을 수 있다. 

3. **제작된 MCP의 결과물 시연** 
  1. MCP에게 남자 이미지 하나를 주고 농구 슛하는 연속 동작 이미지 10장을 만들고, 이를 합쳐 GIF로 만들도록 명령했다. 
  2. 그 결과, 단 몇 분 만에 연속 동작 GIF 결과물이 완전 자동화로 생성되었다. 

4. **제작 방식 예고**: 이 MCP는 **바이브 코딩**만으로 제작될 예정이다. 


## 2. GPT-5와 클로드의 협업을 통한 MCP 개발 준비 


### 2.1. GPT-5를 활용한 자료 조사 및 학습 (딥리 서치) 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/zklpQ5NlSfk/63.jpg)
1. **GPT-5 싱킹 모드 활용** 
  1. GPT-5의 싱킹 모드에서 '딥리 서치' 기능을 선택했다. 
  2. GPT-5에게 "나노 바나나 MCP를 만들 것이니 자료를 만들어 달라"고 명령했다. 

2. **학습 자료 지정 및 목적** 
  1. 가장 중요한 자료인 **나노바나(Gemini 2.5 Flash Image) 공식 홈페이지**를 싹 다 조사하도록 지시했다. 
  2. 다른 이미지 생성 MCP들이 어떻게 만들어졌는지도 함께 조사하도록 했다. 
  3. 이는 단순히 작동하는 것을 넘어 **아주 잘 돌아가는** MCP를 만들기 위함이다. 

3. **딥리 서치 결과 및 저장** 
  1. 딥리 서치에는 약 <mark>20분</mark> 정도 소요되지만, 결과물의 품질은 매우 좋다. 
  2. 딥리 서치 결과는 PDF 파일로 다운로드 가능하며, 이를 다운받아 저장해 둔다. 


### 2.2. 클로드를 활용한 MCP 계획 및 구조 설계 
![image]()
1. **클로드에게 PDF 파일 전달 및 명령** 
  1. 클로드 코드를 실행시키고, GPT-5가 만든 PDF 파일을 클로드에게 전달했다. 
  2. 클로드에게 "이것으로 이미지 만드는 나노바나 MCP를 만들어 달라"고 요청했다. 
  3. 바로 코딩을 시작하는 대신, 먼저 **계획과 구조**를 짜고 중요한 내용은 **지침**으로 만들어 달라고 요청했다. 

2. **클로드의 분석 및 설계 과정** 
  1. 클로드는 GPT-5가 만든 PDF를 분석하여 나노바나 API를 분석하고, 비슷한 MCP들의 구조를 훑어본다. 
  2. 이후 MCP 서버 구조를 설계하고 개발 가이드라인을 작성한다. 
  3. 이처럼 GPT-5와 클로드의 협업을 통해 전체 틀을 간단하게 잡을 수 있었다. 

3. **최종 작업 계획 확인**: 총 <mark>18단계</mark>의 작업 계획이 도출되었으며, 이대로 따라가기만 하면 MCP 제작이 완료된다. 


## 3. MCP 테스트, 디버깅 및 기능 개발 완료 


### 3.1. 초기 에러 발생 및 GPT-5를 활용한 디버깅 꿀팁 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/zklpQ5NlSfk/162.jpg)
1. **초기 테스트 및 에러 발생** 
  1. 17단계까지 완료 후, 마지막 18단계인 테스트를 위해 클로드 코드에 나노바나 MCP를 설치했다. 
  2. 예상대로 설치 직후 에러가 쏟아져 나왔다. 

2. **GPT-5를 활용한 디버깅 방법 (역할 부여 꿀팁)** 
  1. 에러 해결을 위해 GPT-5의 분석 능력을 활용했다. 
  2. **디버깅 절차**:
  1. 소스 폴더 전체를 G 파일로 압축한다. 
  2. 압축 파일을 챗지피티에게 던져주고 에러 분석을 요청한다. 
  3. 간단한 에러는 딥리 서치 없이 GPT-5 싱킹 모델만으로도 좋은 결과를 얻을 수 있다. 
  3. GPT-5는 약 <mark>3분</mark>간 생각한 후 스스로를 **MCP 디버거**라고 소개하며 역할을 정한다. 
  4. **핵심 꿀팁**: AI에게 역할을 정해주면 생각의 범위가 좁혀져 성능이 향상된다는 논문이 많으며, 이처럼 스스로 역할을 정하고 생각하게 하면 결과물이 훨씬 정확해진다. 
  5. 역할을 정하고 오래 생각하게 하는 지침을 사용해도 토큰 사용량은 짧은 생각과 동일한 가치를 가지므로 걱정할 필요가 없다. 
  6. 분석 결과가 나오면 이를 복사하여 클로드 코드에 붙여 넣어 에러 해결에 도움을 받는다. 

3. **까다로운 문제 해결 과정** 
  1. 까다로운 문제는 딥리 서치를 돌려 해결했다. 
  2. **사례**: 나노바나로 만든 이미지가 파일 형식을 지원하지 않는 문제가 발생했다. 
  3. 이 경우에도 전체 소스를 압축하여 챗지피티에게 보내고 **딥리 서치**를 사용하여 해결책을 얻은 후, 그 결과를 클로드 코드에 붙여 넣어 문제를 해결했다. 
  4. 이러한 방식으로 막히는 문제들을 하나씩 해결해 나갔다. 


### 3.2. MCP 수정 및 기능 개발 완료 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/zklpQ5NlSfk/274.jpg)
1. **MCP 재연결 기능 개선** 
  1. MCP는 코드를 바꾸면 새로 고침이 필요한데, 이전에는 프로그램을 껐다 켜야 하는 불편함이 있었다. 
  2. 현재는 클로드 코드 팀의 업데이트 덕분에 /MCP로 나노바나를 찾아서 **재연결**만 누르면 수정 사항을 바로 확인할 수 있게 되어 매우 편리해졌다. 

2. **최종 개발된 MCP의 4가지 주요 기능** 
  1. 멋진 이미지를 생성한다. 
  2. 말 한마디로 이미지를 편집할 수 있다. 
  3. 여러 이미지를 합칠 수 있다. 
  4. 서버 상태를 확인 가능하며, 만든 이미지 개수나 예상 비용까지 알려준다. 

3. **보너스 기능**: 스타일 프리셋 기능이 있어 원하는 분위기를 바로 지정할 수 있다. 


## 4. MCP 활용 사례: 자동 이미지 편집, GIF 제작, 동화책 제작 


### 4.1. 클로드를 활용한 이미지 편집 및 코디 자동화 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/zklpQ5NlSfk/330.jpg)
1. **MCP의 장점**: 복잡한 코딩 없이도 AI(클로드)가 MCP를 자기 손발처럼 사용하여 작업을 수행한다. 
2. **오피스룩 코디 사례** 
  1. 클로드에게 미리 만들어둔 모델 이미지를 주고 오피스룩으로 코디해 달라고 요청했다. 
  2. **조건**: 웹에서 직접 옷 이미지를 찾아서 모델에게 입혀야 한다. 
  3. 클로드는 실제로 브라우저를 켜서 옷을 찾기 시작했고, 금방 두 가지 버전의 오피스룩 이미지를 완성했다. 
3. **구두 코디 추가 사례**: 구두를 신겨 달라고 요청하자, 클로드는 다시 구두를 찾아 화이트 로퍼 스타일과 브라운 로퍼 스타일로 완성했다. 
4. **결론**: 클로드와 MCP의 조합은 클로드가 전자동으로 작업을 처리해 주어 매우 편리하다. 


### 4.2. 가족 사진 생성 및 GIF 연속 동작 제작 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/zklpQ5NlSfk/391.jpg)
1. **10년 후 가족 사진 생성** 
  1. 남자, 여자 사진을 주고, 이 둘이 결혼 후 10년이 흘러 아들, 딸을 낳고 물놀이하는 장면을 그려 달라고 요청했다. 
  2. **조건**: 아들, 딸은 엄마 아빠를 닮게 하고, 엄마 아빠는 10년의 세월이 흐른 느낌이 나도록 했다. 
  3. 클로드는 나노바나 MCP를 호출하여 세부 요청을 영어로 바꿔서 처리했다. 
  4. 완성된 사진은 설명처럼 세월이 흐른 느낌이 나며, 아이들도 부모를 닮은 모습으로 나왔다. 

2. **농구 슛 연속 동작 GIF 제작** 
  1. 나노바나의 **캐릭터 고정 기능**을 활용하여 그럴싸한 GIF를 만들기로 했다. 
  2. 남자 이미지를 주고 농구 슛하는 연속 동작 이미지 10장을 만들고, 이를 합쳐 GIF로 만들도록 요청했다. 
  3. 클로드는 연속 이미지 10장을 만들고 이를 하나로 합쳐 움직이는 이미지(GIF)를 완성했다. 
  4. 결과물은 살짝 어색하지만, MCP로 순식간에 만들어졌다는 점에서 매우 놀랍다. 
  5. 이 작업을 완료하는 데 <mark>5분도 안 걸렸다</mark>. 

3. **독특한 컨셉의 GIF 제작**: 여자가 낙하산을 타고 내려오다가 구름을 만나 장난치는 모습을 GIF로 만들었는데, 독특한 컨셉으로 제작되어 동심으로 돌아간 기분을 느끼게 했다. 


### 4.3. 완전 자동화 동화책 제작 (HTML 파일) 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/zklpQ5NlSfk/482.jpg)
1. **동화책 제작 요청** 
  1. **등장인물**: 남자아이, 여자아이, 강아지, 고양이, 수달. 
  2. **주제**: 동물 친구들과의 신나는 달리기. 
  3. **형식**: 총 <mark>10페이지</mark> 분량의 HTML 파일로 만들어 달라고 요청했다. 
  4. 클로드는 먼저 페이지별 기획을 수행했다. 

2. **제작 결과 및 확인** 
  1. 이미지 10개가 금방 만들어졌으며, 캐릭터들이 모두 똑같이 생겨 **일관성이 매우 뛰어나다**. 
  2. 브라우저로 열어보면 위에는 그림이, 아래에는 이야기가 나오는 형태로 구성되어 있다. 
  3. 글, 그림, 심지어 코딩까지 **전부 클로드에게 맡긴 결과**이며, 매우 편리하다. 


## 5. MCP 배포 및 보안 검토, 설치 방법 안내 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/zklpQ5NlSfk/524.jpg)
1. **MCP 공식 배포 결정** 
  1. 제작된 MCP를 혼자 쓰기 아까워 구독자들과 함께 사용하기 위해 공식 배포하기로 결정했다. 

2. **보안 검토 및 안정성 확보** 
  1. 배포를 위해 **보안**에 가장 신경을 썼다. 
  2. **클로드의 보안 기능 활용**: 최근 클로드에 추가된 /security review 명령어를 입력하여 코드를 뒤져 보안 검사를 수행했다. 
  3. 보안 리뷰 결과를 바탕으로 문제가 될 만한 부분들을 모두 수정했다. 
  4. 이 외에도 여러 번 GPT-5로 딥리 서치하며 보안 안정성을 최대로 높였다. 

3. **소스 코드 공개 및 설치 방법** 
  1. 소스 코드는 신뢰성을 위해 **깃허브에 전부 투명하게 공개**했다. 
  2. 깃허브의 리드미 파일에는 설치 방법부터 사용법까지 친절하게 적혀 있다. 
  3. **설치 절차**:
  1. 영상 댓글에 있는 내용을 모두 복사한다. 
  2. 클로드에 그대로 붙여 넣고, 사용자의 **Gemini API 키**만 추가한다. 
  3. 마지막으로 "나노바나 MCP 설치해 줘"라고 명령하면 설치가 완료된다. 
  4. 이로써 말 한마디로 이미지 생성과 편집을 마음껏 할 수 있게 된다.
