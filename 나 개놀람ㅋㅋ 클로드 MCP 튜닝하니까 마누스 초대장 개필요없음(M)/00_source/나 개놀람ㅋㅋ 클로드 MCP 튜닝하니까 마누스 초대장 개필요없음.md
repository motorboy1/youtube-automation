## 💡 MCP 튜닝은 어떻게 하나요?
 클로드 데스크톱 설정 파일에 특정 내용을 복사해 붙여넣고 재시작한 뒤, 노드 js가 설치된 상태에서 프로젝트 기능을 활용하여 클로드가 MCP 기능을 잘 사용하도록 지침을 제공하면 됩니다. 


클로드 데스크톱에 **MCP(****전문가 도구 세트****)**를 결합하여, 고가의 에이전트 서비스인 '마누스'를 대체할 수 있는 **최강의 AI 연구 에이전트**를 직접 구축하는 실용적인 방법을 공개합니다. 웹 검색, 프로그램 실행 등 33가지 기능을 클로드에 추가하고, **프로젝트 플랜 파일**을 활용하여 장시간의 복잡한 연구 작업도 오류 없이 이어갈 수 있는 **구체적인 튜닝 노하우**를 배울 수 있습니다. 최신 정보에 기반한 심층 보고서 작성부터 웹페이지 제작까지, 클로드의 잠재력을 극대화하여 시간과 비용을 절약하는 **실질적인 AI 활용 전략**을 얻어가세요.
## 1. 클로드 MCP 튜닝을 통한 최강 AI 에이전트 구축 배경 및 결과 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/LrBh_bZSoP8/0.jpg)
1. **클로드 ****MCP 튜닝****의 목적**: 클로드 데스크톱에 **MCP(****전문가 도구 세트****)**를 결합하여 고가의 에이전트 서비스인 마누스(Manus)를 대체하거나 능가하는 AI 연구 에이전트를 직접 구축하는 것이 목적이다. 
  1. MCP는 자동차에 옵션을 달아 성능을 높이는 것처럼, 클로드 데스크톱의 성능을 향상시키는 역할을 한다. 
  2. 여기에 챗GPT의 오퍼레이터나 마누스와 같은 **에이전트 기능**을 추가하는 아이디어를 적용하였다. 

2. **강력한 기능 추가 및 에이전트 탄생**: 클로드에 웹 검색, 구글 검색, 프로그램 실행 기능 등 강력한 부품을 추가한 결과, 기대치를 훨씬 뛰어넘는 **엄청난 에이전트**가 탄생하였다. 
  1. 이 에이전트는 최신 정보에 약한 기존 AI의 한계를 극복하고, 웹 및 구글 검색을 통해 **최신 내용**으로 연구를 수행할 수 있다. 

3. **실제 연구 결과물 시연**: 에이전트에게 연구를 시킨 결과, 컴퓨터에 20개가 넘는 파일 형태의 연구 결과물이 생성되었다. 
  1. **연구 주제**: 최근에 나온 **지브리 스타일 GPT 4 모델**에 대한 심층 연구였다. 
  2. **결과물 형태**: 연구 결과는 웹페이지 형태로 제작되었으며, 이미지 생성 기능 분석, 프로젝트 개요, 기술 분석 파트 등이 포함되었다. 
  3. **기술 분석 내용**:
  1. 벤치마크 점수가 제시되었으며, 다른 모델과의 비교 분석이 이루어졌다. 
  2. 미드저니 V6와 유사하지만, **텍스트 정확도는 더 높다**는 결론이 도출되었다. 
  3. **한계점**도 명확히 찾아냈는데, 처리 속도가 느리고, 다국어 지원 및 특수 시각 효과가 제한적이라는 점이다. 
  4. **보고서의 상세함**: 강점, 약점 설명, 산업별 활용 예시, 프롬프트 전략, 창의적인 활용법, 사례 연구 등 매우 자세하고 정확한 내용이 포함되었다. 

4. **마누스 서비스와의 비교 및 분석**: 직접 마누스를 사용해 본 경험과 마누스가 클로드에 의존한다는 사실을 바탕으로, 튜닝된 클로드의 결과물이 마누스보다 더 좋다고 평가되었다. 
  1. 클로드 3.7은 이처럼 특화된 에이전트로 만들 때 **성능이 월등히 좋아지는 것**으로 확인되었다. 


## 2. 클로드 MCP 튜닝 및 프로젝트 기능 활용 노하우 


### 2.1. 클로드 데스크톱 MCP 설정 방법 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/LrBh_bZSoP8/138.jpg)
1. **핵심 설정 단계**: 클로드 데스크톱에서 개발자 설정을 편집하여 MCP 기능을 추가한다. 
  1. **경로**: 파일 → 설정 메뉴 → 개발자 탭 → 설정 편집 클릭. 
  2. **설정 파일 편집**: 메모장 등으로 설정 파일을 열어 제공된 내용을 복사하여 붙여 넣는다. 
  3. **재시작**: 클로드 데스크톱을 다시 시작하여 설정을 적용한다. 
  4. **윈도우 환경 문제 해결**: 윈도우에서 바로 적용되지 않을 경우, 컴퓨터를 껐다 켜면 해결된다. 

2. **필수 선행 조건**: **노드 JS****(Node JS)** 프로그램이 컴퓨터에 미리 설치되어 있어야 한다. 

3. **추가된 기능 확인**: 설정 후 클로드에 구글 검색, 웹사이트 보기, 프로그램 실행 기능이 생겼으며, 이는 **MCP라는 ****전문가 도구 세트**가 추가된 것이다. 
  1. MCP에는 망치 아이콘과 함께 표시되는 것처럼, 무려 **33가지 기능**이 포함되어 있다. 

4. **폴더 접근 권한 설정**: 설정 파일 하단에 명시된 클로드 프로젝트 폴더(예: 제시 드라이브)를 통해 클로드가 해당 폴더에 파일을 읽고 쓸 수 있는 권한이 부여된다. 


### 2.2. 프로젝트 플랜 파일을 활용한 작업 연속성 확보 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/LrBh_bZSoP8/207.jpg)
1. **프로젝트 기능 활용의 중요성**: 클로드가 새로운 기능을 추가해도 바로 잘 쓰지 못하므로, **프로젝트 기능**을 활용하여 기능을 효과적으로 사용하도록 유도해야 한다. 

2. **비법 지침 설정**: 새 프로젝트를 만들 때, 클로드가 MCP 기능을 잘 사용하도록 **예시들을 지침에 길게 넣어준다**. 

3. **작업 지시 및 플랜 파일 생성**:
  1. 클로드에게 결과물을 넣을 **폴더를 생성**하도록 지시한다. 
  2. **프로젝트 플랜 파일**을 하나 만들도록 지시한다. 
  3. **플랜 파일의 역할**: 이 파일 덕분에 클로드는 해야 할 일, 진행 상황, 다음 단계를 파악할 수 있으며, 프로그램이 꺼지거나 작업이 길어져도 새 창에서 이 파일을 보고 작업을 이어서 할 수 있어 매우 편리하다. 

4. **작업 오류 방지 꿀팁**: 생성되는 파일 하나가 너무 길면 오류가 발생할 수 있으므로, 경험상 **18kb**가 넘으면 여러 개로 나눠서 작업하라는 지침을 추가한다. 

5. **정보의 풍부성 확보**: 정보의 풍부함과 신뢰도를 높이기 위해 **웹사이트 최소 20개 이상**을 찾아서 조사하라는 지침을 넣어 둔다. 


## 3. 심층 연구 미션 수행 및 웹페이지 제작 과정 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/LrBh_bZSoP8/272.jpg)
1. **연구 미션 부여**: 클로드에게 **GPT4 이미지 생성 모델 ****심층 조사** 미션을 상세히 부여한다. 
  1. **추가 요구 사항**: 창의적인 활용법을 직접 생각해 보고, 특정 분야가 아닌 **모든 분야**에서 정보를 찾아보도록 지시한다. 

2. **클로드의 작업 과정**: 클로드는 지시를 받자마자 심층 조사에 착수하며, 다음과 같은 단계를 거친다. 
  1. **정보 수집**: 구글 검색을 시작으로 웹사이트를 돌아다니며 정보를 수집하고, 전문적인 연구 논문과 유튜브 영상까지 찾아본다. 
  2. **정리 및 보고서 작성**: 폴더를 만들어 찾은 자료들을 정리하고, 활용 사례 및 비교 정보를 추가로 찾아 깊이 있는 보고서를 작성한다. 

3. **작업 시간 및 진행 관리**: 이 심층 연구 작업은 솔직히 **세 시간** 정도 소요될 만큼 진심으로 진행되었다. 
  1. **프로젝트 계획 파일**: 계획 파일을 열어보면 해야 할 일들이 상세하게 기록되어 있으며, 끝난 작업은 완료 표시가 되어 있어 꼼꼼하게 진행되었음을 알 수 있다. 

4. **최종 보고서 기반 ****웹페이지 제작**: 최종 보고서가 나온 후, 클로드에게 이 보고서 내용을 바탕으로 **웹페이지를 만들라**는 새로운 미션을 부여한다. 
  1. **웹페이지 제작**** 과정**: 클로드는 즉시 HTML, CSS, 그리고 자바스크립트 파일을 만들어 웹사이트를 구축한다. 
  2. **결과물 확인**: 완성 후 폴더를 열어보면 파일들이 가득 차 있으며, 인덱스 파일을 열어보면 오래 작업한 만큼 **방대한 내용**이 담겨 있음을 확인할 수 있다. 


## 4. 결론 및 활용 가치 평가 
![image](https://resource-release.s3.ap-northeast-2.amazonaws.com/thumbnails/LrBh_bZSoP8/372.jpg)
1. **마누스 서비스 대체 가능성**: 튜닝된 클로드는 마누스 서비스와 매우 비슷하게 작동하며, 클로드가 마누스의 역할을 완전히 수행할 수 있는 것으로 보인다. 
  1. **비용 효율성**: 마누스가 작업 다섯 개에 39달러를 받는 것에 비해, 클로드는 한 달 20달러로 훨씬 많은 일을 할 수 있어 **마누스를 쓸 필요가 없다**는 생각이 든다. 

2. **향후 계획**: 아직 몇 번의 테스트만 진행했으므로 더 많은 사용이 필요하며, 이 활용법을 더욱 업그레이드할 계획이다. 

3. **공유 및 독려**: 이 활용법을 구독자들에게 공개하며, 좋은 결과가 나올 때마다 영상으로 계속 공유할 예정이다.
