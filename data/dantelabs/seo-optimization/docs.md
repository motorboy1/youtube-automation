# SEO Optimizer

## Overview
Maximize your YouTube video discoverability with AI-powered SEO optimization. Research keywords, optimize metadata, and outrank competitors.

## Core Features
- **Keyword Research**: Find high-volume, low-competition keywords
- **Title Optimization**: Generate click-worthy titles under 100 characters
- **Description Generator**: SEO-optimized descriptions with timestamps
- **Tag Recommendations**: Relevant tags for maximum reach
- **Competitor Analysis**: Learn from top-performing videos
- **SEO Score**: Real-time optimization scoring

## Optimization Process
1. **Research Phase**: Keyword analysis and competitor research
2. **Generation Phase**: Create optimized metadata
3. **Refinement Phase**: Iterate based on SEO score
4. **Implementation Phase**: Apply to your videos

## Title Best Practices
- Include primary keyword
- Keep under 60 characters for mobile
- Use numbers and power words
- Create curiosity without clickbait
- Front-load important information

## Description Structure
1. **First 150 characters**: Hook with main keyword
2. **Body**: Detailed video description
3. **Timestamps**: Chapter markers for better UX
4. **Links**: Channel, social media, related content
5. **Hashtags**: 3-5 relevant hashtags

## Tag Strategy
- Start with specific, long-tail tags
- Include broader category tags
- Add branded tags
- Limit to 15-20 most relevant tags
- Research competitor tags
